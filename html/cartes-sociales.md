# Cartes Sociales

- [The Open Graph Protocol](https://ogp.me/)
- [Card Validator Twitter](https://cards-dev.twitter.com/validator)
- [Card Validator Facebook](https://developers.facebook.com/tools/debug)

`HTML`

```html

<html>
  <head>
    <!-- OBLIGATOIRE -->
    <meta property="og:title" content="Le blog personnel de Louis-Nicolas [MAX 70 CHARS]">
    <meta property="og:description" content="Bienvenue sur mon blog... [MAX 200 CHARS]">
    <meta property="og:image" content="https://believemy.com/uploads/mon-image.png">
    <meta property="og:url" content="https://believemy.com">
    
    <meta name="twitter:card" content="summary_large_image">
    
    <!-- ANALYSE / Facultatif -->
    <meta name="twitter:site" content="@BelievemyFR">
    <meta property="fb:app_id" content="app_id">
    
    <!-- IMPORTANTES MAIS EN DEBAT -->
    <meta property="og:site_name" content="Believemy">
    <meta property="og:image:alt" content="Le texte alternatif de l'image">         
  </head>
  
  <body>
  </body>
</html>

```
